# 🛠 Projects

## 🌱 IDIYANALE NPK Monitoring System
During the second semester, my first project was a group project supervised by Kodic Guzman, in which we developed a system using C++ and the Arduino Uno. 

The project was successfully presented during our school exhibit, where we demonstrated our skills in programming, hardware integration, and teamwork.

The NPK Sensor Project is an intelligent agricultural system developed to measure the levels of nitrogen (N), phosphorus (P), and potassium (K) in the soil. These nutrients are essential for healthy plant growth, as they contribute to leaf development, stronger roots, and improved crop production.

The system uses an NPK sensor integrated with an Arduino Uno to gather and display real-time soil nutrient data. Through the analysis of these nutrient levels, farmers and gardeners can identify the proper amount of fertilizer required, helping minimize waste while enhancing crop quality and productivity.

This project demonstrates the integration of technology in modern agriculture by offering a more efficient, accurate, and convenient approach to soil monitoring and nutrient management.
